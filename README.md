# e-Pass-Management-Script
Curfew e-Pass Management system (CPMS) is a e-Pass Management PHP Script which is developed in PHP with MySQL database will manage the records of passes which are issued by administrative office or the state government. This online curfew e-Pass management PHP software which is very helpful at that time curfew or emergency time issued by the state. This e-Pass management system admin has all the options to manage categories, passes, and search options with reports. After the admin generates the pass then people can search their pass using his or her pass id and take print.
